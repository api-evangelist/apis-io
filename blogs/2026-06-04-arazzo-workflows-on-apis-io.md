The APIs.io network introduced a new Arazzo workflows layer this week, indexing the sequences of API calls that providers expose rather than just individual endpoints. Arazzo is the OpenAPI Initiative's specification for describing ordered, multi-step API workflows — the actual sequences of operations needed to accomplish tasks, with inputs, outputs, and branching logic. The initial index pass revealed 4,540 total workflow specifications across 347 providers.
