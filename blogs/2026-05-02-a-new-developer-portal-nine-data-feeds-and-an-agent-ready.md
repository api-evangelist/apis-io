The APIs.io developer portal has been rebuilt from the ground up. The old version documented two HTTP APIs — search and authentication — in a minimal UI that hadn’t changed since 2023. The new portal is organized around a different model: nine static JSON feeds that expose the full APIs.io network as consumable data, a design that matches the main apis.io site, and an agent-readiness layer baked into every subdomain in the network.
